🎯 Implementation Details

  • Components: Created HomeButton.tsx and UndoButton.tsx
  • Removed: GameControls.tsx and HoldToResetButton.tsx
  • Layout: Home above grid, Undo below grid and matches grid width
  • Labels: "Undo Last Move" (EN) / "Letzten Zug rückgängig" (DE)
  • Responsive: clamp() sizing and 48px minimum touch targets
  • Keyboard: Shows (U) shortcut hint when physical keyboard detected

📐 Responsive Features

  • Home Button: 44px min height, responsive padding and font size
  • Undo Button: 48px min height, matches grid width (280px-500px)
  • Touch Targets: All buttons meet accessibility guidelines
  • Font Scaling: clamp(14px, 3.5vw, 16px) adapts to screen size
  • Spacing: Viewport-based margins and padding
